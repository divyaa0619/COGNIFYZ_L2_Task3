# COGNIFYZ_L2_Task3

**⚙️ Task 03 - Feature Engineering**

**🎯 Objective**

To create new features from existing restaurant data and encode categorical variables for analysis.

**🛠️ Tools Used**

   • Python
   
   • Pandas
   
   • Numpy 
   
   • Google Colab


   **📌 Steps Performed**

   1.Loaded the dataset
   
   2.Explored dataset columns
   
   3.Created a restaurant name length feature
   
   4.Created an address length feature
   
   5.Encoded the table booking feature
   
   6.Encoded the online delivery feature
   
   7.Previewed the engineered features


   **💡 Key Insights**

   • Feature engineering improves dataset usability.
   
   • Restaurant name and address lengths were extracted as new features
   
   • Categorical variables were converted into numerical format
   
   • Encoded features help prepare data for machine learning models
